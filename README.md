# GOV.AGI

A satirical government digital service portal for artificial general intelligence, built in React.

**Live site:** https://govtagi.com

## Files

- `index.html` — standalone build (React + Babel + Tailwind via CDN), served by GitHub Pages at [govtagi.com](https://govtagi.com)
- `gov-agi.jsx` — the original React component
- `CNAME` — GitHub Pages custom-domain configuration (`govtagi.com`)

## About

GOV.AGI is a work of satire created by [Bill Chen](https://billchen.online). It is a parody and is not affiliated with, endorsed by, or connected to any real government. The full disclaimer is on the site's **About this website** page.
